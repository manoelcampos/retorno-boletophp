$tipo

$tipo :

Type

Tags

property

Tipo da linha lida do arquivo, conforme valores definidos no layout do arquivo de retorno. Tal valor é uma constante inteira que define se a linha é um header, trailer, linha detalhe, etc. Os valores específicos de tal atributo são definidos nas classes que herdam de @see RetornoBase, em métodos como getIdDetalhe().

$dados :

Tags

property

Vetor associativo contendo os valores da linha lida do arquivo, onde os índices do vetor são os nomes das colunas e o conteúdo de cada posição é o valor da respectiva coluna da linha lida.
